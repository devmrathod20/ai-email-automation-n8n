# 🚀 AI Email Automation using n8n

---

## 📌 Overview

This project is an AI-powered email automation system that processes incoming emails, summarizes them using AI, and sends real-time WhatsApp notifications.

---

## 🔧 Features

* 📩 Gmail trigger for incoming emails
* 🧠 AI-based email summarization
* 📊 Stores data in Google Sheets
* 📲 Sends WhatsApp notifications using Twilio
* ⚡ Priority detection and action extraction

---

## ⚙️ Tech Stack

* n8n (Workflow automation)
* OpenRouter API (AI processing)
* Gmail API
* Google Sheets API
* Twilio WhatsApp API

---

## 🔄 Workflow

Gmail Trigger → AI (OpenRouter) → Data Processing → Google Sheets → WhatsApp Notification
